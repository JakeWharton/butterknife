Configuração no Eclipse

  1. Clique com o botão direito no Explorador de pacotes do seu projeto, vá para Propriedades, navegue para compilador Java → processamento de anotações e marque "Habilitar configurações específicas do projeto".

    Garanta que a outra configuração de processamento de anotações são as mesmas que as mostradas abaixo:

  2. Expanda a seção de Processamento de Anotações e selecione Factory Path. Marque "Habilitar configurações específicas do projeto" e então clique em "Add JARs…". Navegue para a pasta libs/ do projeto e selecione o jar do Butter Knife.

  3. Clique em "Ok" para salvar as novas configurações. O Eclipse irá pedir para fazer o rebuild de seu projeto, você deve clicar em "Sim"
  4. Confira se a pasta .apt_generated/ está na raíz do seu projeto. Ela pode conter arquivos como YOURACTIVITY$$ViewBinder.java. Se estes arquivos não estiverem presentes dê um clean build selecionando Project → Clean. Esta pasta e seus arquivos não devem ser marcados no controle de revisão.
  5. Por último, em "Compilador Java", confira se a versão do java está definido para maior ou igual a 1.6.
